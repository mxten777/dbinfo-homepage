## 🛠️ 기술 스택

### Frontend
- **Next.js 15.5.4** - React 풀스택 프레임워크
- **React 19.1.0** - 사용자 인터페이스 라이브러리
- **TypeScript** - 타입 안정성
- **Tailwind CSS** - 유틸리티 CSS 프레임워크

### 성능 & 최적화
- **Turbopack** - 고속 번들러
- **Dynamic Imports** - 코드 스플리팅
- **Intersection Observer** - 스크롤 애니메이션
- **React Error Boundary** - 에러 처리

### 개발 도구
- **ESLint** - 코드 품질 관리
- **Prettier** - 코드 포매팅
- **Husky** - Git 훅 관리
